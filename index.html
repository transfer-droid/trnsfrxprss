Задать вопрос
Как фиксируется курс?

Сайт получает рыночные данные из открытых API и рассчитывает курс Transfer Express по единой сервисной формуле для всех направлений. Финальный курс и доступность направления подтверждаются оператором перед обменом.

Нужна ли регистрация?

Нет. У Transfer Express нет личного кабинета, паролей и формы регистрации. Всё общение идёт через Telegram.

Какие сети и реквизиты поддерживаются?

Реквизиты зависят от выбранного направления. Для криптовалют это адрес кошелька в нужной сети, для банков — карта или IBAN, для платёжных систем — логин / e-mail / номер счёта, для наличных — детали встречи и выдачи.

Где проходит обмен?

Место и время офлайн-встречи согласовываются с оператором индивидуально. Для безналичных направлений детали также подтверждаются отдельно.

Проводится ли AML/KYC-проверка?

Да, при необходимости оператор может запросить дополнительные сведения или отказаться от операции, если она вызывает обоснованные риск-сигналы.
